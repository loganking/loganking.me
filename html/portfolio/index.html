Blogging for Books

Marketing site for RandomHouse titles. Public bloggers are able to sign up for an account and request books in exchange for posting reviews. As the program has expanded, it was redesigned to better reflect RandomHouse as a whole instead of just a subset of imprints. Features extensive controls for managing book inventory, available, orders, and accounts.

Front-end built on top of Bootstrap and jQuery. Admin control panel uses Prototype. Connects to Klout, ExactTarget, Edelweiss, and NetGalley APIs.

Role: Back-end Development, Front-end Development
